   - Inside try block:
     a. `load_dotenv()`

     g. `collection_name = get_collection_name(pdf_path)`
     h. `embeddings = get_embeddings(provider)`
     i. `llm = get_llm(provider)`
     j. Create PGVector store: `PGVector(embeddings=embeddings, collection_name=collection_name, connection=connection_string, use_jsonb=True)` -- NO pre_delete_collection (read-only)
     k. Create PromptTemplate with template=PROMPT_TEMPLATE, input_variables=["contexto", "pergunta"]
     l. Define inner `_run(inputs: dict) -> str` that:
        - Gets query from inputs["query"]
        - Calls `vector_store.similarity_search_with_score(query, k=10)` (RETR-01, RETR-02)
        - Joins chunk text: `"\n\n".join(doc.page_content for doc, _score in results)` (plain concat, no numbering, no metadata per locked decision)
        - Formats prompt: `prompt.format(contexto=contexto, pergunta=query)` (RETR-03)
        - Calls `llm.invoke(formatted)` and returns `response.content` (RESP-01)
     m. Wrap in `RunnableLambda(_run)` to create chain
     n. If question parameter is provided: invoke chain, print result with console.print, return None
     o. Otherwise: return chain for chat.py to use

4. **Standalone CLI** (`if __name__ == "__main__"`):
   - Parse sys.argv: question from argv[1], optional pdf_path from argv[2] (default "document.pdf")
   - If no question provided: print usage message and sys.exit(1)
   - Call `search_prompt(question=question)`
