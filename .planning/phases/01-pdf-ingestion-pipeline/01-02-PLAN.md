   ```bash

   ```
   Wait for healthy status (check with `docker compose ps`).

2. Verify .env file exists with required vars (OPENAI_API_KEY, DATABASE_URL). If .env doesn't exist, copy from .env.example and remind user to fill in OPENAI_API_KEY.

3. Ensure a test PDF exists. If `document.pdf` doesn't exist in the project root, create a simple one using pypdf's PdfWriter to create a minimal PDF with a blank page. Any valid PDF will work for testing the pipeline.

   ```bash
   python src/ingest.py document.pdf
   ```

5. Verify vectors are stored by querying the database:
   ```bash
   docker exec postgres_rag psql -U postgres -d rag -c "SELECT COUNT(*) FROM langchain_pg_embedding WHERE collection_id IN (SELECT uuid FROM langchain_pg_collection WHERE name = 'document_pdf');"
   ```
   Expected: count > 0

6. Run ingestion a SECOND time to verify the re-ingestion confirmation prompt appears:
   ```bash
   echo "n" | python src/ingest.py document.pdf
   ```
   Expected: Prompts "Collection exists. Clear and re-ingest? (y/n)" then prints "Aborted."
