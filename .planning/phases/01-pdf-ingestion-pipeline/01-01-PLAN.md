**Imports:** sys, os, re, pathlib.Path, dotenv.load_dotenv, PyPDFLoader (from langchain_community.document_loaders), RecursiveCharacterTextSplitter (from langchain_text_splitters), OpenAIEmbeddings (from langchain_openai), PGVector (from langchain_postgres), rich.console.Console, rich.panel.Panel, rich.progress.Progress/SpinnerColumn/TextColumn/BarColumn/MofNCompleteColumn/TimeElapsedColumn, tenacity (retry, stop_after_attempt, wait_fixed, retry_if_exception_type), sqlalchemy (create_engine, text), sqlalchemy.exc.OperationalError

**Functions to implement:**

1. `get_pdf_path() -> str`: Check sys.argv[1], fall back to "document.pdf"

2. `get_collection_name(pdf_path: str) -> str`: Check PG_VECTOR_COLLECTION_NAME env var first. If not set, take Path(pdf_path).name, replace non-alphanumeric with underscore, strip leading/trailing underscores, lowercase.

3. `validate_env()`: Check OPENAI_API_KEY and DATABASE_URL. If either missing, print rich Panel with red border, title "[red]Error[/red]", listing missing vars. sys.exit(1).

4. `test_db_connection(connection_string: str)`: Decorated with @retry(stop=stop_after_attempt(3), wait=wait_fixed(2), retry=retry_if_exception_type(OperationalError)). Creates engine, executes SELECT 1. On final failure, catch and print rich Panel: "Could not connect to database. Is Docker running?" then sys.exit(1).

5. `check_collection_exists(connection_string: str, collection_name: str) -> bool`: Query the langchain_pg_collection table to check if collection_name exists. Use SQLAlchemy engine + text query. Return True/False. Wrap in try/except -- if table doesn't exist, return False.

6. `ingest_pdf()`: The main flow:
   a. load_dotenv()
   b. validate_env()
   c. Get pdf_path via get_pdf_path()
   d. Validate PDF exists -- if not, rich error Panel + sys.exit(1)
   e. Get connection string from DATABASE_URL
   f. Test DB connection (with retry)
   g. Get collection_name
   h. Check if collection exists. If yes, prompt: "Collection '{collection_name}' exists. Clear and re-ingest? (y/n): " using input(). If answer != 'y', print "Aborted." and sys.exit(0).
   i. Load PDF: PyPDFLoader(pdf_path).load()
   j. Split: RecursiveCharacterTextSplitter(chunk_size=1000, chunk_overlap=150).split_documents(documents)
   k. Create embeddings: OpenAIEmbeddings(model=os.getenv("OPENAI_EMBEDDING_MODEL", "text-embedding-3-small"))
   l. Create vector store: PGVector(embeddings=embeddings, collection_name=collection_name, connection=connection, use_jsonb=True, pre_delete_collection=True) -- use pre_delete_collection=True since user already confirmed
   m. Add documents with rich Progress bar:
      - Use Progress with SpinnerColumn, TextColumn, BarColumn, MofNCompleteColumn, TimeElapsedColumn
      - Description: f"Processing {Path(pdf_path).name}"
      - batch_size = 10
      - Loop: for i in range(0, len(chunks), batch_size), add batch, advance progress by len(batch)
   n. Print "[green]Done[/green]"

7. `if __name__ == "__main__": ingest_pdf()` at bottom
